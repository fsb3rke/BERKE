# BERKE

## INTERPRETER COMMAND
```bash
python berke.py hey.br
```

## OUTPUT
```
> B
> E
> R
> K
> E
```

OR YOU CAN USE DEBUG MODE

## INTERPRETER COMMAND
```bash
python berke.py hey.br -d
```

## OUTPUT
```
> 66
> B
> 69
> E
> 82
> R
> 75
> K
> 69
> E
```
